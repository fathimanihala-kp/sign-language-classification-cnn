# 🤟 Sign Language Classification using CNN

## 🔹 Project Overview
This project uses Convolutional Neural Networks (CNN) to classify hand signs from the Sign Language MNIST dataset.

## 🔹 Models Used
- AlexNet
- VGG

## 🔹 Results
- AlexNet achieved higher accuracy
- AlexNet required less training time compared to VGG

## 🔹 Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## 🔹 Dataset
Sign Language MNIST (Kaggle)
